Easily install Tend's tracking script in Wordpress, and start seeing what drives customers.

== Description ==
Tend is the easy way to see what drives customers. Automatically capture website visitor data. Find out which marketing tactics make people take action. And create smart marketing automation workflows based on the data. 


== Installation ==

This section describes how to install the plugin and get it working.  Find detailed instructions here: https://tend.io/docs/Setting-up-Tend/install-tend-on-wordpress

1. Upload the plugin files to the `/wp-content/plugins/tend-tracking-script` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to Settings->Tend Script Install page, and paste in your tacking script in the provided input field.
